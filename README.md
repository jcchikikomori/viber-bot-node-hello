Viber NodeJS bot prototype based from the following:

- Chatgenie PH
- Other Chatbots used in Facebook Messenger

## Setup

### Prequesites

- Viber Account
- Viber Bot's Access Token
- NodeJS
- `pnpm`
- `standard` (standardjs)
- ngrok

### Procedure

1. Run ngrok on a specific port like `8080`
2. Execute `pnpm run index`
